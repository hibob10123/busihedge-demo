BusiHedge Demo
==============

This is a static one-page demo for a small-business fuel hedging product.

What it includes
----------------
- `index.html`: full interactive demo UI (market list, contract flows, portfolio view).
- Live price simulation on hero chart (updates every few seconds).
- `Demo Mode` toggle with additional hover tips and demo sidebar explanations.
- Smooth component transitions and focused mode styling.

How to run
----------
1. Open `index.html` in a browser.
2. Click "Demo Mode" to activate the guided explanation panel.
3. Hover over controls in demo mode for contextual tips.
